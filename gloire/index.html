Vérifier via log

Colle ton report code Fellowship Logs, on vérifie automatiquement tous tes hauts-faits éligibles en une seule passe.

▶ Comment trouver mon report code ?
  1. Va sur fellowshiplogs.com
  2. Cherche ton personnage
  3. Clique sur le donjon concerné
  4. Choisis le report (difficulté / date)
  5. Copie le code dans l'URL

Exemple : https://www.fellowshiplogs.com/reports/LhW2zdPKxtH4cXrC?fight=15...
Le code est LhW2zdPKxtH4cXrC
